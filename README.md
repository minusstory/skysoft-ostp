#### 하늘연 오픈소스 테스트 프로젝트

본 프로젝트는 github의 기본적인 프로세스를 통해
git의 기본적인 사용법을 익히기 위한 프로젝트 입니다
git 사용에대한 예시는 git version 1.9.4.msysgit.2 을 기반으로 제작되었습니다
